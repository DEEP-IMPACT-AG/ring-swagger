swagger-json

(swagger-json swagger)(swagger-json swagger options)

Inputs: ([swagger :- Swagger] [swagger :- Swagger options :- Options])

Produces swagger-json output from ring-swagger spec. Optional second argument is a options map, supporting the following options with defaults:

:ignore-missing-mappings? (false) - whether to silently ignore missing schema to json-schema mappings.

:default-response-description-fn ((constantly "")) - a fn to generate default response descriptions from http status code
